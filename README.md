# Description

The Minesweeper game connects to wss://hometask.eg1236.com/game1

```
help      - returns valid commands
new L     - starts new session, L=1|2|3|4
map       - returns the current map
open X Y  - opens cell at X,Y coordinates
```

# Result

- GitHub repo
- Netlify deployment
